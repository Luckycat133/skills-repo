# agent-skills-setup Development Docs

This folder is reserved for planning, documenting, and iterating on the `agent-skills-setup` skill.

## Purpose

- track future improvements
- capture publishing decisions
- keep development notes separate from the publishable skill files

## Recommended Files

- `roadmap.md`: upcoming features and refactors
- `release-checklist.md`: pre-publish verification steps
- `ideas.md`: experiments and backlog items

## Current Workflow

1. Edit the source skill under Antigravity.
2. Sync it to local IDE installations.
3. Import the updated skill into this repository.
4. Prepare release docs here before publishing.
